# Customer Behavior Analysis (SQL)

## Objective
Analyze customer demographic and spending behavior data using SQL to identify trends nd high-value customer segments.

## Tools Used
- MySQL
- SQL (SELECT, GROUP BY, AVG, CASE, ORDER BY)

## Analysis Performed 
- Counter total customers
- Compared spending behavior by gender
- Calculated average income by demogrpahic groups
- Created age-based customer segments using CASE statements
- Identified the higest spending customer segments

## Key Insights
- Female customers showed higher average spending scores than male customers.
- Customers under 25 had the higesht average spending behavior overall.
- Female customers under 25 represented the higesht spending demographic segment in the dataset.

## Business Recommendations
- Focus marketing campaigns on younger high-spedning customer segments.
- Create personalized promotions targeting valuable demogrpahic groups.
- Use demogrpahic analysis to improve cutomer tareting strategies.
